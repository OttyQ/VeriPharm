Документация VeriPharm

О проекте

VeriPharm — это кроссплатформенная медицинская система, разработанная для повышения безопасности медикаментозной терапии и анализа взаимодействий лекарственных препаратов.

Основные возможности

  • Учёт медикаментов и назначений
  • Анализ лекарственных взаимодействий
  • Идентификация препаратов по штрих-коду
  • Формирование PDF-отчётов
  • Поддержка Android и Windows
  • Облачная база данных (Clever Cloud MySQL)

Архитектура

Система построена с использованием современных технологий:

  • .NET 8 и .NET MAUI для кроссплатформенности
  • Entity Framework Core для работы с данными
  • MySQL (облачное хранение) для синхронизации

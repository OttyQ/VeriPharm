## 🧭 Обзор

**VeriPharm** — это кроссплатформенная медицинская система, разработанная для повышения безопасности медикаментозной терапии и анализа взаимодействий лекарственных препаратов.

---

## ✨ Основные возможности

- 💊 Учёт медикаментов и назначений  
- ⚠️ Анализ лекарственных взаимодействий  
- 📦 Идентификация препаратов по штрих-коду  
- 📄 Формирование PDF-отчётов  
- 🖥️ Поддержка Android и Windows  
- ☁️ Облачная база данных (Clever Cloud MySQL)  

---

## 🧩 Проблема

Ошибки при приёме и назначении лекарств остаются одной из наиболее распространённых и предотвратимых проблем в здравоохранении.

VeriPharm решает её за счёт:

- структурированного хранения данных  
- автоматического анализа взаимодействий  
- унифицированного интерфейса

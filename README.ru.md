## 🧭 Кроссплатформенная система управления медикаментозной терапией и анализа взаимодействий лекарств

**VeriPharm** — это современное медицинское приложение, разработанное на **.NET 8 и .NET MAUI**, предназначенное для повышения безопасности при приёме лекарств, анализа лекарственных взаимодействий и ведения структурированных медицинских данных.

---

## ✨ Основные возможности

- 💊 Учёт медикаментов и медицинских назначений  
- ⚠️ Анализ лекарственных взаимодействий  
- 📦 Идентификация препаратов по штрих-коду  
- 📄 Формирование PDF-отчётов  
- 🖥️ Поддержка Android и Windows  
- 🧠 Современная архитектура на .NET  

---

## 🧩 Проблема, которую решает система

Ошибки при назначении и приёме лекарств являются одной из самых распространённых и предотвратимых проблем в здравоохранении.

VeriPharm решает эту задачу за счёт:

- структурированного хранения медицинских данных  
- автоматического анализа взаимодействий  
- простого и понятного интерфейса  

---

## 🏗️ Архитектура

VeriPharm построен на модульной .NET архитектуре с облачным хранением данных:

- .NET 8 (ядро приложения)
- .NET MAUI (кроссплатформенный интерфейс)
- Entity Framework Core (слой доступа к данным)
- MySQL (облачная база данных на Clever Cloud)
- разделение бизнес-логики и интерфейса

---

## ⚙️ Технологический стек

| Слой | Технология |
|------|------------|
| UI | .NET MAUI |
| Backend | .NET 8 |
| ORM | Entity Framework Core |
| База данных | MySQL |
| Отчёты | генерация PDF |
